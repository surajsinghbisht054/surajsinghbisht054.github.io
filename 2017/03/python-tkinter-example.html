Q 1. What Is Tkinter?


Ans. Tkinter is python's standard GUI (Graphical User Interface) package Used for creating Graphic in 

python. Tkinter is  a thin object-oriented layer on top of Tcl/Tk. Tkinter Is Not Only One GUI Package 

Which Can Be used for creating cross-platform. graphic interface, their are many more available on 

internet like wxpython, pygame and many more. but important thing is that Tkinter is Standard Gui 

package of python that's why tkinter is normally pre installed in python.



so,  let's talk about some features of python Tkinter module .

  • Easy To Code
